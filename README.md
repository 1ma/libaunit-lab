# libaunit-lab

A toy project to explore the [AUnit](https://github.com/AdaCore/aunit) testing framework.

## Build

`$ gprbuild -f -p -P harness`

## Run

```
$ ./bin/harness

OK Calcul Unit Tests : Test Suma function
OK Calcul Unit Tests : Test Divisio function

Total Tests Run:   2
Successful Tests:  2
Failed Assertions: 0
Unexpected Errors: 0
```
